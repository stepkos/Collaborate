# Collaborate
Platform to serch people to projects!

## How to run?
- Install PHP and MySQL
- Create empty database

- Execute this file
```
db/init.sql
```

- Fill data to connect with database here
```
db/config.php
```

```php
return [
    'host'      =>  '<your host>',
    'user'      =>  '<your username>',
    'password'  =>  '<your password>',
    'database'  =>  '<your database>'
];
```
